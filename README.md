# Im celebrity

## 😎  서포터즈 선정 어드민 사이트
- [바로가기](https://dev-cmkim.github.io/im-celebrity)
- 신청자들의 정보를 보고 해당 프로젝트에 선정자로 선택 및 취소 할수 있는 서비스 입니다.

## 💻  기술 스택 
 -  React
 -  Scss
 -  json-server


## 기능 
    ⭐️  선정자는 최대 3명까지 선택 가능 -> 선택시 하단 버튼에 선정인원 표시
    ⭐️  리스트 내 전략 드롭다운 효과
    ⭐️  브랜드 참여내용 확인 모달 구현
    ⭐️  선정하기/취소하기 클릭시 확인 팝업 구현
